# Setup Requrement
1. Install the virtual environment

```bash
python3 -m venv env
```
2. Activate the virtual env below command for (mac)

```bash
source env/bin/activate
```
3. Most needed library to install for work

```bash
pip3 install numpy
pip3 install sklearn
```
